wrangler.api

Automator

object Automator

Automator can make changes to a list of git repos, such as applying a shell script to each repo or updating a versions of artifacts, commit those changes and raise pull requests for each of the repos.

Starting points are runAutomator to apply a shell script or runUpdater to update versions.

  23. def parseUpdaterConfig(path: String): Updater[Config]

    Parse the config for the version updater.

    Parse the config for the version updater.

    It needs to specify a list of repos to update and a list of artifacts whose versions it should change. Instead of a target version the version could be latest. This can then be resolved to a specific version number in resolveArtifact.

    An example is:

    targets = [wrangler, maestro]
artifacts = [
  au.com.cba.omnia % edge   % 2.1.0-20140604032756-0c0abb1,
  au.com.cba.omnia % tardis % latest,
  org.apache.sqoop % sqoop  % 1.4.3-cdh4.6.0
]
